# Agents

## Overview

Архитектура, паттерны и типы AI-агентов. Раздел охватывает фундаментальные концепции построения агентных систем: от простых tool-use агентов до сложных multi-agent оркестраций.

## Structure

```
agents/
├── agents.md           # Этот файл — навигация по разделу
└── ...                 # Будущие документы по конкретным темам
```

## Topics

- **Tool Use** — как агенты вызывают внешние инструменты и обрабатывают результаты
- **Memory** — типы памяти: in-context, файловая, векторная, эпизодическая
- **Planning** — ReAct, Chain-of-Thought, Tree-of-Thought и другие паттерны рассуждений
- **Multi-Agent** — оркестрация, специализация, изоляция контекста между агентами
- **Permissions** — модели разрешений, безопасность, blast radius действий
