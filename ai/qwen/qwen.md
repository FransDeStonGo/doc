# Qwen (Alibaba Cloud)

## Overview

Qwen — семейство моделей от Alibaba Cloud. После DeepSeek стал вторым громким open-source успехом из Китая. Qwen2.5 серия покрывает широкий спектр: от мультимодальных до специализированных code и math моделей. Активно используется как база для дистилляций других моделей (в т.ч. DeepSeek R1 Distill).

## Models

| Модель | Контекст | Роль в агентной системе |
|--------|----------|-------------------------|
| Qwen2.5-72B-Instruct | 128k | Основной агент: конкурирует с GPT-4o по бенчмаркам |
| Qwen2.5-Coder-32B | 128k | Code-агент: один из лучших open-source для кода |
| Qwen2.5-VL-72B | 128k | Vision-агент: анализ документов, скриншотов, UI |
| QwQ-32B | 32k | Reasoning-модель: конкурент R1 при меньшем размере |
| Qwen2.5-7B / 3B | 128k | Edge и мобильные агенты, локальный деплой |

## Strengths for Agents

- **Open weights** — большинство моделей полностью открыты, доступны на HuggingFace
- **Qwen2.5-Coder** — один из топовых open-source code-агентов, превосходит DeepSeek Coder на ряде задач
- **QwQ-32B** — reasoning при скромном размере, запускается на одной A100
- **Мультимодальность** — Qwen-VL нативно понимает изображения, документы, UI-скриншоты
- **Alibaba Cloud** — API через DashScope, интеграция с экосистемой Alibaba
- **База для дистилляций** — DeepSeek R1 Distill-Qwen модели используют Qwen как backbone

## Considerations

- Как и DeepSeek — китайская инфраструктура, важно для enterprise privacy
- Документация и комьюнити преимущественно на китайском для новых фич
- API через DashScope менее зрелый чем OpenAI/Anthropic по экосистеме интеграций
